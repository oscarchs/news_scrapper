Para esta función de WhatsApp, se debe tener en claro que solo se podrán encontrar los mensajes no borrados de la cuenta, ni tampoco las conversaciones eliminadas. Los buscadores únicamente permiten encontrar la palabra que aún están presentes en los archivos del chat de conversaciónPara encontrar mensajes nuevos o antiguos, se debe utilizar los buscadores que WhatsApp tiene incorporados. Cuando el usuario buscar en específico un mensaje de una conversación, en ocasiones el mensaje puede ser muy antiguo, y encontrarlo tomara mucho tiempo. Por ese motivo, en esta guía se describirá paso a paso el procedimiento para encontrarlo de forma rápida y sencilla.¿Como buscar los chats usando una palabra clave?En esta sección se explicará el proceso para encontrar una palabra en los chats de conversación.En primer lugar, se debe abrir WhatsAppDespués, dirigirse a los chats.Posteriormente, ubicar el buscador de chat, que se encuentra en la parte superior. Dar clic en el buscador y escribir la palabra o frase que se requiere buscarPor último, aparecerán varios resultados de la búsqueda, solo se debe pulsar en el mensaje del chat para que se abra.¿Como buscar archivos multimedia con filtros?Se debe seguir las siguientes indicacionesPara empezar, el usuario se debe abrir la aplicación de WhatsAppDespués, se debe ir a los ChatsSeguidamente, se debe ubicar la barra de búsquedaLuego, se abrirá un menú desplegable, que contiene diferentes formatos como fotos, videos, audio, documentos, se deberá seleccionar el archivo.Posteriormente, se debe escribir la palabra clave que se asocie con la imagen o video que se quiere encontrar.Por último, se mostrarán diversos resultados, se debe tocar el deseado para que el elemento multimedia se abra.Te recomendamos METADATA, el podcast de tecnología de RPP.
