El usuario de PC, busca manejarla de forma versátil y de la manera más cómoda posible, con herramientas que le hagan más fácil la vida a la hora de tener una experiencia con este ordenador. Por ello, a ciencia cierta, todas las personas que tienen una computadora a la mano saben las herramientas más básicas, desde apagar y prender la PC, entrar a internet o crear un documento y escribir en el. Estos conocimientos, son de los más comunes en el mundo informático y nada difíciles de aprender. Sin embargo, para un púbico que busca innovarse constantemente y tratar de trabajar con inmediatez, existen distintos atajos para lograr con mayor efectividad e inmediatez su labor. Por ello, aquí algunos comandos para lograr efectuar atajos de forma más rápida.Comandos rápidosCTRL + RETROCESOSirve para eliminar una palabra completa. Tan solo el usuario debe presionar la tecla CTR y retroceder simultáneamente, de este modo logrará eliminar una fila de palabrasALT + TABSirve para cambiar a otra ventana abierta. Tan sólo debe mantener la tecla ALT+TAB. De este modo cambiará de forma inmediata la ventana que desee trabajar.CTRL + WCon este atajo el usuario podrá cerrar varios sitios a la vez. CTRL + LAl mantener presionado la tecla CTRL+L podrá resaltar la barra de direcciones de la web, tras ello podrá escribir una dirección web de forma inmediata.AgitarSi el usuario tiene varios programas abiertos y desea quitarlos, debe usar el cursor para agitar la ventana activa. Esto minimizará automáticamente todos los demás programas que se encuentren abiertos.WINDOWS + DSi el usuario desea minimizar todos los programas, incluso en el que se encuentra en ese momento, debe presionar Windows+D. CTRL + TEste truco sirve para abrir una pestaña a la hora de navegar en Google.WINDOWS + LPara poder bloquear una computadora en la cual, el usuario se encuentre trabajando, debe dar click a WINDOWS+L, desde modo la pantalla se bloqueará y protegerá el dispositivo.Te recomendamos METADATA, el podcast de tecnología de RPP.
