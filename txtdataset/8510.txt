Toda PC posee una dirección IP que representa la ubicación en Internet del dispositivo. La dirección IP está presente en los dispositivos que mantienen una conexión a Internet, llegando a poseer una dirección IP privada y una dirección IP pública.En la presente guía se explicará paso a paso cómo encontrar la dirección IP privada o pública de una PC de una forma sencilla.¿Qué es una dirección IP pública?Está conformada por 3 cifras en una fila de 4, estos números están separados por puntos. Esos códigos permiten que el dispositivo sea reconocido y pueda navegar en Internet.¿Cómo encontrar una dirección IP pública?Únicamente se deben de seguir las siguientes indicaciones:Primero, se debe ingresar a la interfaz del routerDespués, mirar la dirección IP pública, que normalmente está a la vista.¿Qué es una dirección IP privada?En cambio, la dirección IP privada permite que los dispositivos conectados en un mismo router se comuniquen entre sí. Por ejemplo, si se conecta un celular a un router, se le asignará una dirección IP privada.¿Cómo encontrar una dirección IP privada?Sistema WindowsPrimero, pulsar el icono Windows + R.Se abrirá un el cuadro de diálogo Ejecutar.Después, escribir en el cuadro el siguiente código: ipconfig.Por último, aparecerá un cuadro con la dirección IPv4.Otra forma de conocer la dirección IP privada.Ir a Configuración, luego, hacer clic en la opción Red e Internet.Después, selecciona Estado que se encuentra en las opciones del lado izquierdo de la pantalla.En la parte derecha de la pantalla se encontrará la opción Propiedades de la red activa, solo pulsar sobre ella.Por último, dentro de esa opción se visualizará la dirección IP privada.Sistema iOSSe debe de realizar los siguientes pasos.Abrir la ventana Preferencias del Sistema y pulsar en la opción Red.En la opción Red seleccionar el tipo de conexión que se está usando.Por último, ir a la opción Estado del recuadro central, ahí indicar la dirección IP privada.Te recomendamos METADATA, el podcast de tecnología de RPP.
