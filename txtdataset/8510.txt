Toda PC posee dirección IP representa ubicación Internet dispositivo. dirección IP presente dispositivos mantienen conexión Internet, llegando poseer dirección IP privada dirección IP pública.En presente guía explicará paso paso cómo encontrar dirección IP privada pública PC forma sencilla.¿Qué dirección IP pública?Está conformada 3 cifras fila 4, números separados puntos. códigos permiten dispositivo reconocido pueda navegar Internet.¿Cómo encontrar dirección IP pública?Únicamente deben seguir siguientes indicaciones:Primero, debe ingresar interfaz routerDespués, mirar dirección IP pública, normalmente vista.¿Qué dirección IP privada?En cambio, dirección IP privada permite dispositivos conectados mismo router comuniquen sí. ejemplo, si conecta celular router, asignará dirección IP privada.¿Cómo encontrar dirección IP privada?Sistema WindowsPrimero, pulsar icono Windows + R.Se abrirá cuadro diálogo Ejecutar.Después, escribir cuadro siguiente código: ipconfig.Por último, aparecerá cuadro dirección IPv4.Otra forma conocer dirección IP privada.Ir Configuración, luego, hacer clic opción Red Internet.Después, selecciona encuentra opciones lado izquierdo pantalla.En parte derecha pantalla encontrará opción Propiedades red activa, solo pulsar ella.Por último, dentro opción visualizará dirección IP privada.Sistema iOSSe debe realizar siguientes pasos.Abrir ventana Preferencias Sistema pulsar opción Red.En opción Red seleccionar tipo conexión usando.Por último, ir opción recuadro central, ahí indicar dirección IP privada.Te recomendamos METADATA, podcast tecnología RPP.
