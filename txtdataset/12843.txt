Si bien participantes divididos si pensaban aplicaciones chat accedían micrófonos silenciadas, mayoría creía aplicaciones deberían poder recopilar datos mientras silenciadas.Para segunda parte estudio, equipo investigó comportamiento real botón silencio muchas aplicaciones populares, determinando tipo datos recopilan si podrían revelar información personal.Utilizaron herramientas análisis binario tiempo ejecución rastrear audio procesar aplicaciones videoconferencia populares medida audio viajaba aplicación controlador audio computadora luego red mientras aplicación silenciada.Descubrieron todas aplicaciones probaron ocasionalmente recopilan datos audio procesar mientras silencio activado, aplicación popular recopila información entrega datos servidor misma velocidad, independientemente si micrófono silenciado no.Luego, investigadores decidieron ver si podían usar datos recopilados silencio aplicación inferir tipos actividades lugar segundo plano.
