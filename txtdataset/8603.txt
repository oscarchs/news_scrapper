El nuevo sistema Windows 11 ha traído consigo nuevas mejoras, como lo es el widget del clima, esta opción está ubicada en la barra de tareas. Como otro widget de clima, este muestra, la temperatura, clima y la zona actual de la ubicación.Cuando se hace clic en el icono del widget, este se expande mostrando todas sus funciones, sin embargo, normalmente, se presiona casualmente y el que aparezca en la pantalla puede ser incómodo. Es por ello, que en la presente guía se explicará cómo eliminar el widget del clima en Windows 11.Las diferentes formas de eliminar el Widget del Clima en Windows 11Si se realiza los siguientes pasos se eliminara de forma rápida el widget del clima en el sistema Windows 11.Mediante la configuración de WindowsPrimero se debe abrir la configuración de Windows, para ello, se usa las teclas Windows + IDespués, ir a Personalización y luego, a Barra de TareasUbicar la función de WidgetsPor último, pulsar desactivarMediante el uso de RegistroEjecutar los siguientes procedimientos.Para empezar, se debe abrir el indicador de ejecución con las teclas de Windows + RSeguidamente, se debe escribir regedit y presionar Shift + Enter para que se abra el editor con permiso de administradorDespués, se debe seguir la siguiente ruta: Computadora\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\MicrosoftHacer clic derecho y crear una carpeta DshPosteriormente, crear un nuevo DWORD con el nombre AllowNewsAndInterestsLuego, se debe hacer doble clic para que se abra, luego, seleccionar Hexadecimal y establecer el valor en 0Por último, pulsar sobre el botón Aceptar para guardar los cambios.Mediante la Política de GrupoEjecutar los siguientes indicaciones.Abrir el cuadro de Ejecutar, para ello, efectuar las teclas Windows + RDespués, se debe escribir gpedit.msc y presionar EnterSe abrirá el Editor de políticasLuego, ir a Configuración de la computadoraSeguidamente a las Plantillas administrativas.Posteriormente a Componentes de Windows.Finalmente, se llegará a Widgets y se debe seleccionar deshabilitar para eliminarlo o apagarlo.Te recomendamos METADATA, el podcast de tecnología de RPP.
