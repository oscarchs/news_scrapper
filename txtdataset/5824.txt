Los conectores de USB que permiten a los dispositivos conectarse a la computadora; sin embargo, para sacarlos del equipo es necesario cerrar los procesos que se tiene acceso a sus archivos. Para este proceso se usa la función de Windows Desconexión segura. Al usar esta opción el sistema detiene de forma automática las aplicaciones o procesos que se estaban leyendo o escribiendo en el dispositivo, aunque parezca que el usuario no lo esté usandoSon muchas las aplicaciones o programas que se encuentran en segundo plano, como el antivirus. Es recomendable esta práctica si es que haz realizado antes este tipo de función que permite Windows 10.
