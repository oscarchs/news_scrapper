WhatsApp sabe que hay aun camino por andar en la dura competencia de la mensajería mediante celulares. Una de las condiciones más valoradas por los usuarios es la seguridad y privacidad de los chats, y es ahí en donde los delincuentes se centran: en vulnerar la plataforma para “hacerte creer” que no pasa nada. Ahora, WhatsApp Web añade un nuevo nivel de seguridad con la extensión “Code Verify”.“A diferencia de una aplicación móvil descargable, una aplicación web generalmente se entrega directamente a los usuarios, sin que un tercero revise ni audite el código”, señala Meta en un post. “Hay muchos factores que podrían debilitar la seguridad de un navegador web que no existen en el espacio de la aplicación móvil, como las extensiones del navegador”.WhatsApp añade verificador de código webEn colaboración con Cloudflare, WhatsApp ha creado extensiones para los navegadores web Chrome y Edge – hay planes futuros para Firefox – que revisan el código al ejecutar WhatsApp Web. Esta extensión valida si el sitio es auténtico o si tiene alguna manipulación o brecha. Si detecta una alteración, el usuario recibirá una notificación en la barra de herramientas.Este sistema funciona con tres luces para cada condición hallada al analizar el código:Si el código web de WhatsApp está completamente validado, el ícono de Code Verify en el navegador aparecerá en verde.Si aparece de color naranja, necesitas actualizar la página. También puede ocurrir que otra extensión del navegador está interfiriendo con Code Verify, por lo que se te recomendará pausar las otras extensiones para un descarte.Si aparece en rojo, se ha detectado un posible problema de seguridad con el código web de WhatsApp que estás usando.Cada notificación cuenta con una asistencia para los pasos a ejecutar después, y permitir que el usuario reanude sin problemas la actividad de manera segura.Code Verify es de código abiertoMeta señala que este sistema está disponible para desarrolladores y terceros en GitHub, y puede ser usadas por otras empresas o personas que quieran añadir este sistema adicional de verificación a sus propios recursos web.“Creemos que, con Code Verify, estamos explorando un nuevo territorio con la verificación automática de códigos de terceros, particularmente a esta escala”, señala WhatsApp. “Esperamos que más servicios utilicen la versión de código abierto de Code Verify y conviertan el código web verificado por terceros en la nueva norma. Y al hacerlo, esperamos que esto ayude a brindar protecciones de seguridad adicionales a personas de todo el mundo y haga avanzar a toda la industria”.
