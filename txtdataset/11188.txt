Repositorio Único Nacional Información Salud (Reunis) Ministerio Salud cuenta indicador clave: cuántas dosis vacuna COVID-19 administran diario. Expertos señalado decisión afecta transparencia políticas salud pública.La plataforma Reunis ofrece información serie indicadores relacionados vacunación, porcentaje primera, segunda tercera dosis, avance cada región, grupos etarios, etc. embargo, notado figura gráfico número total vacunas COVID–19 aplicadas cada día.TransparenciaEl médico especialista estadística investigación salud pública, Percy Mayta-Tristán, indicó decisión preocupante, debido afecta transparencia. “Como ciudadanos tener capacidad monitorear avance vacunación y, si quitan algún indicador, va permitir saber proceso vacunación”, manifestó.Asimismo, reflexionó puede retroceder avanzado relacionado datos abiertos. “En general, pandemia permitido tener oportunidad tener datos abiertos ciudadanía haga vigilancia diferentes acciones venido tomando pandemia. podemos retroceder ello. Debemos seguir avanzando tener mayor información pública disponible”, indicó.Para licenciada exdirectora Inmunizaciones Ministerio Salud, Gabriela Jiménez, buscaría decisión “mostrar datos forma positiva”.“El modificar gráfico tan importante velocidad vacunación llama atención. dice pretendiendo mostrar ahora curva cómo acumulando datos. [...] Si hacemos cálculos, solo mirando tablero tercera dosis, pesar manipulado denominador, fecha superamos 50% tres dosis. Si revisamos tablero país, brecha vacunación aún acentuada regiones, cuales deberíamos seguir pensando ayuda técnica”, sostuvo especialista.Jiménez señala claro vacunación desacelerado, ministerio salud debe tomar cuenta recomendaciones. “Sincerar acciones tácticas debemos seguir desarrollando mejorar resultados”, dijo.La especialista señaló clave mejorar percepción población, respecto principal medida protección sanitaria vacuna. “Existe cantidad suficiente vacunas regiones disponen ellas. Ahora necesita empuje. ver tema comunicacional, participación regiones cobertura baja. trata empoderar municipios gobiernos regionales asuman rol protagónico”, manifestó.Por parte, analista datos Juan Carbajal estimó sorpresa revisar cifras respecto dosis aplicadas día sido retirada. “Esos gráficos desaparecieron único mostraban gráfica forma acumulada, gráfico dice ritmo vacunación tal”, expresó.“El mostrar información habla mucho, sesgo. Esperemos sido adrede intención tapar críticas dado conocer bajada ritmo vacunación”, dijo Carbajal.El ingeniero manifestó importancia contar datos abiertos. “Son valiosos toma decisiones, cifras vidas medio. [...] aras eso, importante transparencia, mostrar información. críticas sirven fortalezca pueda ver solución camino”, manifestó.RespuestaEduardo Ortega Guillén, jefe equipo despacho ministerial Ministerio Salud, dijo RPP Noticias gráfico lineal Reunis útil 'para tomar decisiones', reemplazado nuevos muestra curvas diferenciadas, según tipo vacuna: tercera dosis adultos vacunas niños.
