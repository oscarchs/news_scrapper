WhatsApp﻿ es una de las aplicaciones más utilizadas alrededor del mundo debido a que conecta a los usuarios en tiempo real mediante mensajes, llamadas de voz y audios.El anclado de chats en WhatsAppLa plataforma de mensajería instantánea posee interesantes funciones a la hora de chatear con un contacto, tales como: Notas de voz, conversaciones archivadas, opción para subir archivos multimedia, entre otras.Una función muy útil es la de anclado de conversaciones ya que permite diferenciar las conversaciones importantes en medio de todos los chats sin importar su antigüedad.Fijar una conversación es muy sencillo y se realiza en menos de dos minutos, el proceso se realiza desde la misma aplicación y no se necesita de ninguna funcionalidad extra.Pasos para utilizar la función de anclado de chatsPara utilizar la función el usuario deberá contar con la última actualización de WhatsApp esta se puede comprobar ingresando desde Google Play Store o App Store dependiendo del dispositivo móvil y comprobando si el botón actualizar este activo, si la aplicación no necesita actualizaciones entonces se podrá continuar con el proceso: Se debe ir hacia la aplicación y abrirla en la pantalla principal, ahí estarán los últimos chatsRecordemos que WhatsApp los ordena por antigüedad, se debe buscar la conversación con el contacto que se quiera anclarUna vez ubicada se debe posicionar el dedo en la conversaciónDeslizar suavemente de izquierda derecha, la opción contraria para archivar un chat.Aparecerá un ícono de pin, se deberá presionar aquel ícono para fijar un chat al tope de las conversaciones.¡Listo! de esta manera el chat quedará fijado, WhatsApp permite un número limitado de 3 conversaciones fijadas. Cabe recordar que se puede repetir el mismo proceso en los chats, pero a la inversa para quitar esta configuración, se podrán fijar nuevas conversaciones de manera indefinida y cuántas veces el usuario desee.Te recomendamos METADATA, el podcast de tecnología de RPP.
