Los usuarios de Windows están furiosos. La nueva actualización del sistema operativo de Microsoft posee muy buenas características, pero bajo el colchón piensan colocar una función obligatoria. Al igual que la edición Windows 11 Home, la versión Pro requerirá una conexión a Internet y una cuenta de Microsoft durante la configuración.En la versión actual de Windows 11, aún puedes crear una cuenta de usuario local durante la configuración de tu PC, algo que también funcionó en la versión Home de Windows 10, pero que ahora se eliminó en la versión Windows 11 Home. Al menos que Microsoft cambie de parecer, la creación de cuenta local ya no estará disponible.Esto no sería problema para aquellos usuarios que cuentan con Internet y cuenta de Microsoft, pero la tecnología es global y algunos usuarios están furiosos por la nueva condición de Windows 11 Pro.Deberás tener una cuenta Microsoft, quieras o noLa nueva función de Windows 11 Pro hará más difícil donar o vender computadoras, crear imágenes de cientos de sistemas, algo que requieren algunas empresas, o configurar una computadora para otra persona.Además, esta nueva característica le da cierto poder a Microsoft al recopilar cientos de miles de datos sobre más usuarios.'La parte más molesta de esto es que, como propietario de un taller de reparación de PC, mis técnicos ahora tienen que obtener la información de la cuenta MS de las personas solo para reiniciar/reformatear una computadora ahora. O necesitamos que el cliente ingrese y se registre, luego podemos terminar la reparación'. señaló un usuario de Reddit.Te recomendamos METADATA, el podcast de tecnología de RPP.
